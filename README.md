# Personal Finance Dashboard

An interactive dashboard for tracking expenses and analyzing spending patterns.

## Overview
This project processes transaction data using Python and visualizes insights through Power BI dashboards.

## Features
- Expense tracking by category and month
- Data cleaning and transformation using Pandas
- Interactive visualizations in Power BI

## Technologies
- Python (Pandas)
- Power BI
- Microsoft Excel

## What I Learned
- Preparing real world data for visualization
- Designing dashboards to communicate financial insights
